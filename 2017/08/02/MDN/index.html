内容安全策略（CSP）

CSP是一个出色的 WEB 安全方案

简而言之，内容安全策略（Content Security Policy，简称 CSP）是一种以可信白名单作机制，来限制网站中是否可以包含某来源的资源。资源类型细分为 Script、Style、Image、Font、Connect、Media、Object、Frame 等，可分别配置其来源。默认配置下不允许执行内联脚本和内联样式，也禁止执行 eval。
